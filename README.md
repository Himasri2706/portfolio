# Himasri - AI/ML Engineer Portfolio

A professional, recruiter-optimized portfolio website built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customizing Content

All data (projects, skills, experience, contact details) is centralized in a single file for easy editing.
Open `src/lib/data.ts` and look for the `TODO` comments to fill in your personal links, remaining projects, and work experience.

To add your resume, place your `resume.pdf` file inside the `public/` directory.

## Deployment

This project is optimized for zero-config deployment to Vercel.

1. Push your code to a GitHub repository.
2. Go to [Vercel](https://vercel.com/) and import the repository.
3. Click **Deploy**. Vercel will automatically detect Next.js and configure the build settings.
